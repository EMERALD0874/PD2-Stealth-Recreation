# PD2-Stealth-Recreation

## Overview
A recreation of detection in PAYDAY 2's stealth.

## Controls
W - Move forward
A - Move left
S - Move backward
D - Move right
Shift - Sprint
Space - Jump
F - Interact
Mouse - Aim
Mouse 1 - Shoot
